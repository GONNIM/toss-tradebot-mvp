## 1. 사용자 본인의 경험담 (원문 보존)

> 저는 미국 주식 수익률이 100% 이상이지만 아직도 차트나 지표에 대해서는 잘 모릅니다. 그런데도 수익이 났었던 방법 알려드릴게요.
>
> 저는 우선 제가 주식공부를 잘 못한다는 걸 인정을 했습니다. 막 저기 뭐 재야에 막 숨어져 있는 저평가 주식을 찾아야 큰 수익을 낼텐데 그렇게는 못하겠더라고요. 그래서 그냥 누가봐도 좋은 주식 중에서 좀 싼 걸 사보자 이렇게 마음을 먹었습니다.
>
> 그때 당시에 누가 봐도 좋은 주식이 뭘까라고 생각을 해보니까 그때는 미국의 빅테크 기업이 막 뜰 때여서 빅7안에 드는 주식을 사려고 했어요. 엔비디아, 테슬라, 아마존, 메타 이런 것들 7개를 이야기하는 거거든요.
>
> 이런 마음가짐이었어요. 만약에 물려도 기다리면 무조건 오를 우상향할 주식을 고르자. 그리고 그 7개 중에서 최근 가장 많이 떨어진 걸 찾아서 사려고 했습니다.
>
> 가장 많이 떨어진 거 찾는 방법 굉장히 쉬운데 토스 커뮤니티에 들어가서 욕이 가장 많이 달린 거. 요즘에는 마소나 이런 아이들. 이렇게 커뮤니티 분위기 안 좋은 걸 사서 계속 들고 기다리다가 10%, 20%가 올라오면 팔고 하는 걸 굉장히 소액으로 자주 반복을 해서 소소하게 돈을 벌었었어요.
>
> 근데 이렇게 1, 2년 해보니까 빅테크라는 아이들은 제가 판 가격보다 무조건 엄청 비싸지더라고요. 시간이 지나면. 그래서 점점 쌀 때 매수하고 매도는 안하는 방식으로 주식을 모아가기 시작했고 이렇게 매도를 최소화하는 방식으로 투자를 하려다 보니까 점점 매도를 안 할 수 있는 주식을 찾게 되잖아요?
>
> 그러다 보니 점점 지수로 넘어가게 되어서 지금은 지수에 투자하는 비율이 굉장히 높은 편입니다. 소소하게 단타로 투자할 때보다 이런 식으로 떨어졌을 때 매수하고 매도를 하지 않는 방식으로 주식을 모아가다 보니까 그제서야 시드랑 수익이 불어나는 속도가 붙기 시작하더라고요.
>
> 요즘은 개별은 진짜 100만 원 단위로 포모 안 올 정도로만 매수를 하고 거의 다 지수만 사고 있습니다.

## 2. 사용자 전략 단계 분해

| 단계 | 행동 | 의도 |
|---|---|---|
| **자기 인식** | 차트·지표 약점 인정. 저평가 발굴 능력 없음 인정 | 본인 한계를 룰로 메꾸자 |
| **종목 선정** | "누가 봐도 좋은" 빅7 (NVDA, TSLA, AMZN, META 등) | 망할 일 없는 자산만 |
| **매수 신호** | 토스 커뮤니티 욕(부정 댓글) 가장 많은 종목 | 군중 비관을 역지표로 |
| **초기 매도** | 10-20% 오르면 매도, 소액 빠른 반복 | 작은 수익 누적 |
| **진화 1** | 1-2년 후 깨달음: 빅테크는 결국 판 가격보다 훨씬 비싸짐 | 기회비용 인지 |
| **진화 2** | "쌀 때 매수 + 매도 안 함" → 모아가기 | 시드+수익 가속 |
| **진화 3** | 매도 안 할 수 있는 자산 = 지수로 자연 이동 | Buy-and-Hold 본질 |
| **현재** | 개별 = 포모 안 오는 소액(~100만원), 대부분 지수 ETF | 심리적 안정 + 수익 안정성 |

**핵심 원칙 3개**:
1. **신뢰 가능한 자산만** (개별: 빅테크 / 코어: 지수)
2. **군중 비관 = 매수 신호** (sentiment 역지표)
3. **매도 최소화, 매수 집중** (DCA 변형 — 가격 하락 시 분할 매수)

## 3. 전설적 사상 계보 — 5인

| 인물 | 한 줄 철학 | 원전 | 사용자 행동과의 연결 |
|---|---|---|---|
| **Warren Buffett** | "위대한 회사를 합리적 가격에 사서 영원히 보유" — 남이 두려워할 때 탐욕하라 | Berkshire 주주서한 (1965~) / *The Essays of Warren Buffett* | 빅테크 = 위대한 회사 / 욕댓글 = 두려움 |
| **John C. Bogle** | "건초더미에서 바늘을 찾지 말고 건초더미를 사라" | *The Little Book of Common Sense Investing* (2017) | 지수 ETF = 건초더미 (사용자 진화 정점) |
| **Benjamin Graham** | "안전 마진 + 미스터 마켓(조울증 환자)을 활용하라" | *The Intelligent Investor* (1949) | 부정 분위기 매수 = 미스터 마켓 활용 |
| **Howard Marks** | "이류적 사고 (Second-level thinking)" — 모두가 사는 걸 사지 마라 | *The Most Important Thing* (2011) / Oaktree 메모 | 욕먹는 종목 매수 = 역발상의 정확한 원리 |
| **Charlie Munger** | "Sit on your ass investing" — 좋은 거 사놓고 가만히 있어라 | *Poor Charlie's Almanack* | 매도 최소화 = 사용자 진화 단계 |

→ **사용자는 자신을 "공부 못한다"고 했으나, 행동 패턴은 위 5인의 핵심 사상을 본능적으로 수렴함.** 우연이 아니라 시장의 "단순한 진리"에 도달한 것.

## 4. 검증된 저빈도 기법 — 7선

각 기법: 학술 출처 / 검증 기간 / 접근성 ★ / 사용자 매핑

### A. Dollar-Cost Averaging (정액 분할 매수, DCA)
- **원리**: 고정 금액·고정 주기 매수. 가격 무관.
- **검증**: 1979년 Constantinides 논문 이후 수많은 backtest. lump-sum 대비 평균 약간 열위 / drawdown·심리 우위.
- **접근성**: ★★★★★
- **사용자 매핑**: 자동화 베이스라인. 단 "쌀 때 더 사기" 직관과는 약간 어긋남.

### B. Value Averaging (밸류 평균, Edleson 1991)
- **원리**: 자산가치 목표선 정한 후 미달분만큼 매수, 초과분 매도. DCA의 자연스러운 진화형.
- **검증**: Edleson, *Value Averaging* (1991, 2007 개정). IRR이 DCA보다 평균 1~2%p 우위.
- **접근성**: ★★★★ (월 1회 계산 필요)
- **사용자 매핑**: **"떨어지면 더 산다"의 수학적 정량화 — 사용자 직관과 가장 일치**.

### C. Mebane Faber 10-Month Moving Average (TAA)
- **원리**: 자산이 10개월 이평선 위면 보유, 아래면 현금. 4개 자산(미주식·해외주식·채권·원자재) 각각 적용.
- **검증**: Faber 2007, *A Quantitative Approach to Tactical Asset Allocation* — SSRN 다운로드 1위 논문, 70년 데이터 검증. CAGR 유사하나 **drawdown 절반** (S&P -50% 시 시스템 -20%).
- **접근성**: ★★★★★ (월 1회 점검)
- **사용자 매핑**: 추세 추종이라 "쌀 때 사기"와 반대. 단 큰 위기(2008·2020·2022) 회피 효과 검증됨. **방어용 옵션**.

### D. Dual Momentum (Antonacci 2014)
- **원리**: 절대 모멘텀(자산 12m 수익률 > 채권) + 상대 모멘텀(미국 vs 해외 중 우위)
- **검증**: Antonacci, *Dual Momentum Investing* (2014). NAAIM 어워드 수상. AQR/CFA 다수 인용.
- **접근성**: ★★★★ (월 1회, 3개 ETF로 구현)
- **사용자 매핑**: 추세 추종 — 사용자 직관과 다소 거리. 다만 매우 검증된 저빈도 시스템.

### E. Lazy Portfolios — 3-Fund / All-Weather / Permanent
- **원리**:
  - **3-Fund** (Bogleheads): VTI 60% + VXUS 30% + BND 10%
  - **All-Weather** (Ray Dalio): 주식 30% + 장기채 40% + 중기채 15% + 금 7.5% + 원자재 7.5%
  - **Permanent** (Harry Browne): 주식·장기채·금·현금 25%씩
- **검증**: Bogleheads.org 위키 + Portfoliovisualizer 수십 년 데이터. Dalio *Principles*. Browne *Fail-Safe Investing* (1999).
- **접근성**: ★★★★★ (분기 또는 연 1회 rebalance)
- **사용자 매핑**: 매도 최소화 + 지수 비중 큰 사용자 진화의 자연 종착지.

### F. Buy-The-Dip 체계화 (Just Keep Buying)
- **원리**: 사전 정의 임계(예: ATH -10/-20/-30%)에서 분할 매수. 임계 더 깊을수록 매수량 증가.
- **검증**: Nick Maggiulli, *Just Keep Buying* (2022) — DCA vs Buy-The-Dip backtest. **장기 IRR 차이 작음**, 단 **심리적·전략적 정렬은 우수**.
- **접근성**: ★★★★★
- **사용자 매핑**: **사용자 본인 전략의 직접 자동화**. 빅테크 ATH 추적 + 단계별 임계 매수.

### G. CAPE Ratio Global Value Rotation (Shiller / Faber)
- **원리**: Shiller CAPE (P/E 10년 평균)가 낮은 국가/시장에 비중 배분. 매년 또는 분기 rebalance.
- **검증**: Shiller 2013 노벨상 수상 논문. Faber, *Global Value* (2014). CAPE 5분위 하단 시장이 상단보다 장기 수익률 우위 — 100년 데이터.
- **접근성**: ★★★ (CAPE 데이터 분기 갱신, 글로벌 ETF 다수 필요)
- **사용자 매핑**: "지수 + 싸게 사기" 의 글로벌 확장판. 사용자가 한국 종목까지 확장 시 고려.

## 5. 사용자 행동 ↔ 기법 매핑 (정량화)

| 사용자 진화 단계 | 본능적 행동 | 가장 가까운 학술 기법 |
|---|---|---|
| 1단계 (초기) | 빅7 + 욕먹는 거 매수 + 10-20% 익절 | **Value Averaging + Howard Marks 역발상** |
| 2단계 (전이기) | 빅7 + 쌀 때 매수, 매도 거의 안 함 | **Buy-The-Dip 임계화 + Buffett/Munger** |
| 3단계 (현재) | 지수 비중 ↑ + 개별은 포모 안 오는 소액 | **Bogle 인덱스 + Core-Satellite + Lazy Portfolio** |

→ 사용자는 무의식적으로 **Howard Marks 역발상 → Buffett 보유 → Bogle 인덱스** 순으로 진화. **3대 거장의 정확한 발자취**.

## 7. 권고 조합 — 3 후보

### 🥇 후보 1 (강추) — Core-Satellite: Buy-The-Dip + Bogle Index
> "사용자 진화 경로를 그대로 시스템화"

- **Core (80%)**: VOO 또는 QQQ — 매월 정액 DCA, 매도 안 함 (Bogle/Munger)
- **Satellite (20%)**: 빅7 개별 (NVDA·TSLA·AMZN·META 등 사용자 선택) — ATH 대비 단계별 매수
  - -10%: 1차 매수 (시드의 5%)
  - -20%: 2차 매수 (시드의 7%)
  - -30%: 3차 매수 (시드의 8%)
- **매도 정책**: Core 0, Satellite는 사용자 모드 선택 (+20% 익절 OR 영구 보유)
- **빈도**: 월 1회 DCA + 임계 도달 시 자동 매수 (이벤트 기반)
- **철학**: Buffett(보유) + Bogle(지수) + Marks(역발상) + Munger(매도 최소)
- **사용자 매핑**: 1→2→3단계 모두 자연스럽게 포괄

### 🥈 후보 2 — 학술 안전형: Mebane Faber TAA
> "위기 회피 + 검증된 학술 기반"

- 4개 자산 (VOO + VEA + BND + GLD) 각각 10개월 이평선 ON/OFF
- 월 1회 점검
- **장점**: 2008·2020·2022 같은 큰 위기 자동 회피
- **단점**: 사용자 직관 "쌀 때 매수"와 반대 (추세 추종이라 떨어지면 빠짐)
- **사용자 매핑**: 약함

### 🥉 후보 3 — 균형형: All-Weather + DCA
> "Dalio 사상 — 모든 경제 계절에 견디는 포트폴리오"

- Dalio All-Weather 비중 자동 유지
- 매월 정액 DCA
- 분기 rebalance
- **장점**: 변동성 최소, 매도 거의 없음
- **단점**: 빅테크 비중 낮음 → 사용자 빅테크 선호와 거리감
- **사용자 매핑**: 보수적·은퇴 자금 성격이라면 적합

## 9. 잔여 결정 사항 (PRD 작성 전 확정 필요)

### Q1. 자동화 대상 모드
- **모드 A** (단타): 빅테크 개별, 10-20% 익절, 매수→매도 사이클
- **모드 B** (보유): 빅테크 + 지수, Buy-and-Hold, 매수만
- 후보 1 선택 시 → A+B 둘 다 (Core-Satellite 구조로 자동 결정)
- 후보 2/3 선택 시 → B 중심

### Q2. "부정 분위기" 정량화 방식
- (a) 가격 drawdown (ATH 대비 -X%) — 가장 단순·신뢰
- (b) 200일 이평선 대비 -X%
- (c) z-score (변동성 정상화)
- (d) 외부 sentiment API (Reddit/X/StockTwits)
- (e) (a)+(d) 결합
- 후보 1 선택 시 → (a) 권고 (Buy-The-Dip 임계화의 표준)
- 후보 2 선택 시 → 이평선 기반이라 (b)/(c) 무관
- 후보 3 선택 시 → DCA 기반이라 별도 정량화 불필요

### Q3. 매도 정책
- 후보 1 모드 A 부분: +20% 익절 OR 트레일링 OR 시간 기반 — 선택 필요
- 후보 1 Core / 후보 2 / 후보 3 → 매도 거의 없음 (rebalance 외)

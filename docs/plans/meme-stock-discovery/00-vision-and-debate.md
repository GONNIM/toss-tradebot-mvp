## 1. 밈주의 공통 5요소 (사례 분석에서 도출)

전 세계 밈주 폭등 사례를 분해하면 다음 다섯 가지가 거의 항상 함께 등장한다.
한 가지만으로는 부족하고, 동시 발현(confluence)이 핵심.

| # | 시그널 | 정의 | 측정 가능성 |
|---|---|---|---|
| ① | **공매도 잔고 폭증** | 유동주식 대비 short interest 비율 ≥ 15% | 공식 (KRX 일별 공시) |
| ② | **소셜 모멘텀 surge** | 24시간 언급량 z-score ≥ +3σ | Reddit/X/네이버종토방 크롤링 |
| ③ | **유동성 폭주** | 거래량 z-score ≥ +5σ vs 20일 평균 | 공식 (실시간) |
| ④ | **가격 oversold + 반전** | RSI ≤ 30 → +5% 반등 동시 거래량 증가 | 공식 (실시간) |
| ⑤ | **catalyst event** | 거래 정지(VI), 공시, 인플루언서 발언 | 부분 공식 + 크롤링 |

> Wendy's 사례 매핑: ① 공매도 23% (✓) · ② WSB 바이럴 (✓) · ③ 거래량 20배
> (✓) · ④ 5년간 -66% 빠진 상태에서 반등 (✓) · ⑤ 거래 정지 발동 (✓)
> — 5/5 confluence.

## 2. 시장 선택 (가장 큰 결정 분기점)

| 옵션 | 장점 | 단점 |
|---|---|---|
| **A. KRX 단독** | 토스트레이드 본질 (한국 주식), 자료 일관, 공매도 데이터 공식 | 한국 시장에 "밈주" 문화 약함 — 대신 "테마주/작전주" 변형으로 적용 |
| **B. 미국 NASDAQ/NYSE 추가** | WSB·X·Yahoo 등 풍부한 소셜 시그널, 학습 표본 (GME/AMC/BBBY/Wendy's) 풍부 | 백엔드 데이터 소스 이중화, 사용자가 토스로는 미국 주식 매매 못함 |
| **C. 미국만** | 시그널 품질 최상 | toss-tradebot 정체성 이탈 |

**전문가 의견**: **B 권장** — 한국은 시그널 양이 절대적으로 적어 모델
학습 불가능 수준. 미국 사례로 모델을 학습/검증하고, 한국은 "변형 시그널"
(작전주·테마주·정치주) 별도 트랙으로 운영. 다만 **사용자 매매 가능 여부와
무관하게 "정보 서비스" 성격**으로 노출하면 가치 있음.

## 3. 한국 시장의 "밈주 변형"

한국엔 GME/Wendy's 같은 WSB 스타일 squeeze는 거의 없다. 대신:

| 패턴 | 트리거 | 추적 가능성 |
|---|---|---|
| **정치 테마주** | 대선·정치 이벤트, 후보 친·인척 관련주 | 뉴스 NER + 키워드 매칭 |
| **작전주 (pump & dump)** | 시총 작은 코스닥 + 거래량 급증 + 특정 카페·텔레그램 | 거래량 + 공시 패턴, 작전세력 알림방 크롤링 |
| **유증/감자 후 반등** | 자본조정 공시 후 거래 재개 시 단기 폭등 | 공시 모니터링 |
| **AI/2차전지 등 단발 테마** | 거시 뉴스 (예: NVDA 실적, EV 정책) → 한국 관련주 군집 매수 | 뉴스 + 섹터 매핑 |
| **품절주 (low float)** | 유통주식 적은 코스닥 + 거래량 폭증 | KRX 공식 데이터 |

→ KRX 트랙은 "테마/작전 워치"로 별도 운영. 미국 밈주와 다른 모듈로 분리.

## 4. 시그널 소스 — 가용성 매트릭스

| 소스 | 비용 | 안정성 | 추천도 |
|---|---|---|---|
| **Reddit API (PRAW)** | 무료 (rate limit 100 QPM) | 공식 API · 안정 | ⭐⭐⭐ |
| **X (Twitter) API** | 유료 ($100+/월 Basic) | 공식 · 안정 | ⭐⭐ (비용 부담) |
| **Stocktwits API** | 무료 (제한적) · 영문 종목 sentiment | 공식 · 안정 | ⭐⭐⭐ |
| **네이버 종목토론** | 무료 (크롤링) | 변경 risk | ⭐⭐ |
| **디시인사이드 주식갤** | 무료 (크롤링) | 변경 risk | ⭐⭐ |
| **KRX 공매도 잔고** | 무료 (공식 일별 공시) | 매우 안정 | ⭐⭐⭐ |
| **DART 공시** | 무료 (공식 OpenAPI) | 매우 안정 | ⭐⭐⭐ |
| **FINRA short interest (미국)** | 무료 (격주 발표) | 공식 · 안정 | ⭐⭐⭐ |
| **NASDAQ/NYSE 거래량** | 무료 (Yahoo Finance, FinanceDataReader) | 안정 | ⭐⭐⭐ |
| **Google Trends** | 무료 | 안정 | ⭐⭐ |

**1차 도입 권장**: Reddit + Stocktwits + KRX 공매도 + 야후 파이낸스
거래량 + Google Trends. **X는 후속**.

## 5. Cadence — 갱신 주기

| 모드 | 의미 | 트레이드오프 |
|---|---|---|

**1차 권장**: **5분 batch + 종가 후 일배치 (이중)**. 실시간 알림은 후속.

## 6. 출력 형태

| 옵션 | 설명 |
|---|---|
| **A. Meme Score (0~100)** | 5요소 confluence 가중합. 60 이상 = 워치 |
| **B. Squeeze Risk Tag** | 🔥/⚠️/💤 3단계 라벨 + 핵심 시그널 1줄 요약 |
| **C. 워치리스트 페이지** | Top 10 형태, 5요소 시각 표시 |
| **D. 알림 (push)** | 임계값 돌파 시 (후속) |

**1차 권장**: **A + B + C 동시 제공**. D는 검증 후.

다르고, 사용자 의사결정 흐름도 다름 (Moonshot은 1주 보유, Meme은
분/시간 보유).

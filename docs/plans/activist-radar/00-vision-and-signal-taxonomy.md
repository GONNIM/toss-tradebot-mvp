# 00. 비전 · 초기 신호 분류

## 1. "초기"의 정의 — 단계별

Activist 활동은 최소 6개월 걸린 물밑 매집 → 5% 넘어서 SC 13D 신고 → 경영권 요구·이사회 개편 → proxy fight → 결과. "초기"는 이 흐름에서 어느 단계를 잡는가에 따라 다르다.

| 초기성 등급 | 단계 | 감지 방법 | 이번 스코프 |
|---|---|---|---|
| **Level 1** — 매집 진행 중 | 아직 5% 안 넘음 · 옵션 축적 · 물밑 매수 | 옵션 UOA · 거래량 급증 · Peer 사례 확산 · 잔업 트래킹 | **Phase D+** (선행 신호) |
| **Level 2** — 매집 완료 · 첫 공개 | SC 13D 신규 · 대량보유 신규 | SEC EDGAR / DART 즉시 폴링 | **⭐ Phase A~B** |
| **Level 3** — 태세 변경 | 13G → 13D 전환 · 목적 변경 | 필링 폼 전환 감지 | **Phase D** |
| **Level 4** — 지분 강화 | 13D/A · 대량보유 변동 (증가) | 필링 변화 감지 | **Phase A** 부분 커버 |
| **Level 5** — Wolf Pack 형성 | 여러 activist 30일 내 동일 종목 진입 | 시간 window 매칭 | **⭐ Phase C** |
| **Level 6** — 이사회 요구 · Proxy | PREM14A / DEFC14A · 이사회 개편 | 8-K · 필링 폼 | 후속 (매매 결정에는 이미 늦음) |

**핵심 통찰**: 학술 연구(Brav et al. 2008, Coffee & Palia 2016)에 따르면 SC 13D 신고 후 30일 평균 초과수익 **+6~8%**, 1년 초과수익 **+8~12%**. Level 2 진입 자체가 유효한 진입 시그널.

## 2. 신호 유형 분류 (미국)

### 2.1 강도 등급

| 서식 | 신호 강도 | 이유 |
|---|---|---|
| **SC 13D 신규** | ⭐⭐⭐⭐⭐ | 경영권 관심 명시 · 첫 공개 |
| **SC 13G → SC 13D 전환** | ⭐⭐⭐⭐⭐ | passive → active 태세 명확 변경 |
| **13D/A 지분 증가 (>1%)** | ⭐⭐⭐⭐ | 확신 강화 · 매집 지속 |
| **SC 13D 유명 activist** | ⭐⭐⭐⭐⭐ | tier 1 fund 는 실행력 검증됨 |
| **PREM14A / DEFC14A** | ⭐⭐⭐⭐ | proxy fight 개시 · 이미 공개 국면 |
| **13D/A 지분 감소** | ⭐⭐ (반대 신호) | 활동 종료 · 매도 시작 |
| **Form 13F 신규 편입** | ⭐⭐ | 45일 지연 · 매집 확인 정도 |

### 2.2 서식 상세

- **SC 13D**: 5%↑ 지분 취득 시 10일 이내 신고 (경영 개입 목적)
- **SC 13G**: 5%↑ 지분 취득이지만 passive (재무 투자자용)
- **SC 13D/A · SC 13G/A**: 수정본 · 1% 이상 변동 시마다
- **Form 4**: 임원·이사·10%↑ 주주 매매 · T+2 영업일
- **Form 13F**: 100M$↑ 운용사 분기 보고 · 분기 종료 45일

## 3. 신호 유형 분류 (한국)

### 3.1 강도 등급

| 서식 | 신호 강도 | 이유 |
|---|---|---|
| **대량보유 신규 (경영참여 목적)** | ⭐⭐⭐⭐⭐ | US SC 13D 대응 |
| **대량보유 변경 (단순투자→경영참여)** | ⭐⭐⭐⭐⭐ | 태세 전환 |
| **대량보유 변동 (지분 증가)** | ⭐⭐⭐⭐ | 매집 지속 |
| **주주총회 소집청구권 행사** | ⭐⭐⭐⭐ | 실행 국면 진입 |
| **임원·주요주주 소유상황보고** | ⭐⭐⭐ | Form 4 대응 |
| **대량보유 변동 (지분 감소)** | ⭐⭐ (반대) | 매도 시작 |

### 3.2 필수 필터

- **`보유목적` = "경영권에 영향을 주기 위한 것"** — 핵심 필터. 단순 투자 신고는 노이즈
- **filer 이름 매칭** — Universe([[01-universe]]) 에 등재된 activist 만
- **의결권 있는 주식 기준** — 우선주 지분은 신호 강도 낮음

## 4. 강도 스코어링 (Phase C 상세)

각 신호에 0~100 강도 점수:

```
score = base_form_score
      × activist_tier_multiplier
      × market_cap_bonus         (마이크로캡·소형주 신호 강화)
      × wolf_pack_bonus          (30일 내 다른 activist 동일 종목)
      × momentum_bonus           (지분 증가 속도)
```

- **base_form_score**: SC 13D 신규 = 80, 13G→13D 전환 = 90, 13D/A 지분 증가 = 60, 등
- **activist_tier_multiplier**: Tier 1 (Elliott/Icahn/Pershing) = 1.2, Tier 2 = 1.0, Tier 3 = 0.9
- **wolf_pack_bonus**: 30일 내 두 번째 activist = ×1.3, 세 번째 = ×1.5
- **momentum_bonus**: 지분 증가율 상위 25% = ×1.15

**임계값**:
- 80+ 🌋 CRITICAL — 즉시 Telegram + UI 최상단 강조
- 60~79 🔥 STRONG — Telegram
- 40~59 ⚠️ WATCH — UI 만
- <40 참고

## 5. 성공 지표

| 지표 | 목표 |
|------|------|
| 신규 SC 13D 감지 지연 | < 15분 (SEC 필링 후 · 폴링 5분 기준) |
| 대량보유공시 감지 지연 | < 10분 (DART API 반영 속도 고려) |
| 오검출률 (false positive) | < 10% (filer 이름 mismatch 등) |
| 감지 성공 사례 3개 이상 축적 | Phase A~C 완결 후 30일 |
| 개인 매매 활용 사례 발생 | Phase A~C 완결 후 60일 |

## 6. 이번 스코프에서 다루지 않는 것

- **매집 진행 중 신호** (Level 1) — 옵션 UOA · unusual volume 은 별도 Phase
- **뉴스·소셜 activist letters** — Bloomberg 유료 · 스코프 외
- **주주환원 요구·자사주 매입 압력** — 정성 신호 · 텍스트 분석 필요 · 후속
- **Proxy fight 결과 추적** — 이미 늦은 국면
- **KRX 소액주주 결집 사이트** — 정보 신뢰성 낮음

## 7. 위험

- **SC 13D 신고 후 즉시 급등하면 이미 시장 반영** — 신고 감지 후 1일 이내 진입 필수
- **False signal**: activist 가 단기 트레이딩용으로 진입했다가 조기 청산
- **한국 규제 특성**: 5영업일 이내 신고 → 실제 매집 시점 3~5일 지연
- **CIK 이름 매칭 오류** — Tier 리스트 검증 필수

## 4. 알려진 한계 (B-2e 시작 전 인지)

1. **24개월 표본 작음** (23 unique months): Pearson 통계 신뢰도 한계. 더 긴 시계열은 과거 motir PDF 추가 다운 또는 KOSIS API 확장 필요.
2. **DART 수출비중 미통합**: export_ratio_hint 는 임시값. 정확한 DART 사업보고서 자동 추출은 후속 sub-Phase.
3. **매핑 사용자 검수 미완**: 51 종목 코드/이름은 자동 검증 통과했으나 본업 비중·중복 종목(삼성전자=반도체+컴퓨터+가전+무선통신) 처리는 사용자 도메인 지식 필요.
4. **FDR Marcap 단위 표시**: 삼성전자 1,954조원 표시 (실제 약 400조의 4~5배) — UI 표시 시 환산 검토 필요. log10 비교에는 영향 없음.
5. **APScheduler 매월 1일 11:30 KST 자동 동작**: 다음 1개월 후 첫 검증 가능. KDI_NUM_CATALOG 신규 발표 num 수동 갱신 정책 유지.
